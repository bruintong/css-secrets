  • box-shadow：设置盒模型阴影

CSS3 背景

  • background-image：设置背景图像
  • background-size：设置背景图片的尺寸
  • background-origin：设置背景图片的定位区域
  • background-clip：设置背景的绘制区域

CSS3 渐变

  • linear-gradient：线性渐变
  • radial-gradient：径向渐变
  • repeating-linear-gradient：重复线性渐变
  • repeating-radial-gradient：重复径向渐变

CSS3 文本效果

  • text-shadow：文本阴影
  • box-shadow：盒子阴影
  • text-overflow：指定如何显示溢出内容
  • word-wrap：允许长的内容可以自动换行
  • word-break：单词拆分换行

CSS3 字体

  • @font-face

CSS3 2D转换

  • transform：适用于2D或3D转换的元素
  • transform-origin：允许您更改转化元素位置

CSS3 3D转换

  • transform：适用于2D或3D转换的元素
  • transform-origin：允许您更改转化元素位置
  • transform-style：规定被嵌套元素如何在 3D 空间中显示。
  • perspective：规定 3D 元素的透视效果。
  • perspective-origin：规定 3D 元素的底部位置。
  • backface-visibility：定义元素在不面对屏幕时是否可见。

CSS3 过渡

  • transition：简写属性，用于在一个属性中设置四个过渡属性
  • transition-property：规定应用过渡的CSS属性名称
  • transition-duration：规定过渡效果花费的时间
  • transition-delay：规定过渡效果延迟时间
  • transition-timing-function：规定过渡效果的时间曲线

CSS3 动画

  • @keyframes：规定动画
  • animation：所有动画属性的简写属性，除了animation-play-state属性

CSS3 多列

  • column-count：指定了需要分割的列数
  • column-gap：指定了列与列间的间隙
  • column-rule：column-rule-*所有属性的简写
  • column-rule-color：指定了两列的边框颜色
  • column-rule-style：指定了列与列间的边框样式
  • column-rule-width：指定了两列的边框厚度
  • column-span：指定元素要跨越多少列
  • column-width：指定了列的宽度

CSS3 用户界面

  • resize：指定一个元素是否应该由用户去调整大小
  • box-sizing：允许以确切的方式定义某个区域的具体内容
  • outline-offset：设置轮廓框架在 border 边缘外的偏移

CSS3 框大小

  • box-sizing：允许以确切的方式定义某个区域的具体内容

CSS3 弹性盒子

  • display: 指定 HTML 元素盒子类型（flex指定弹性盒子）
  • flex-direction：指定了弹性容器中子元素的排列方式
  • justify-content：应用在弹性容器上，把弹性项沿着弹性容器的主轴线对齐
  • align-items：设置弹性盒子在侧轴方向上的对其方式
  • flex-wrap：指定弹性盒子的子元素换行方式
  • align-content：用于修改flex-wrap属性的行为。类似于 align-items, 但它不是设置弹性子元素的对齐，而是设置各个行的对齐。
  • margin: auto：完美的居中
  • order：设置弹性盒子的子元素排列顺序
  • align-self：设置弹性元素在自身轴方向上的对齐方式
  • flex：指定弹性子元素如何分配空间
